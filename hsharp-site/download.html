安装指引+

macOS

打开 .dmg → 拖入「应用程序」文件夹 → 启动时若提示「未识别开发者」，到「系统设置 → 隐私与安全性」点击「仍要打开」。

Windows / Linux

当前仅发布 macOS 版本 · Windows 与 Linux 移植正在路上，订阅 GitHub Issue 以便第一时间收到发布通知。

macOS 包管理器
brew install hsharp
从源码构建（任一平台） 
git clone https://github.com/peddlejumper/HSharp && cd HSharp
h# build compiler/main.hto -o h#
