常见 疑问

还有问题？写信给我们，通常 24h 内回复。

ZZW Code 的 AI 额度用完了怎么办？ +
不会突然停用 —— 额度耗尽后会自动切换到你填入的自有 API Key（如果你已经填了的话）。没填自有 Key 的用户可以单独购买额外额度包，5 元 / 3 次。
我用 ZZW Code 必须用 H# 团队提供的模型吗？ +
不需要。所有方案都支持「自带 API Key」—— 任何 OpenAI 兼容的 Endpoint（OpenAI / DeepSeek / 通义千问 / 自部署 vLLM 等）都可以直接接入。H# 团队不接触你的 Key 也不留存任何对话。
订阅版的 xxx 次 / 月是怎么算的？ +
一次「AI 对话」= 一次问答回合（你发问 + AI 回复算一次）。续写 / Tab 补全不计额度。每个月 1 号 0 点（UTC+8）自动重置。
Pro 版可以中途取消吗？ +
可以。随用随停 —— 已支付的部分会一直可用到你当前计费周期结束，结束后自动降回 Free 版，不会自动续费。
Team 版的私有模型路由是什么意思？ +
团队管理员可以在 ZZW Code 后台配置一个或多个内部模型（你公司自己的 LLM、经过安全审计的私有部署等），并按团队成员或项目粒度路由特定请求走不同模型。所有路由配置都经过审计日志。
MAX / MAX+ 和 Pro / Team 有什么本质区别？ +
MAX 系列是为「重度 AI 用户」准备的：更大的月度额度（500 / 2,000 次）、旗舰模型优先调度（GPT-5、Claude 4、自部署模型）。MAX+ 在此基础上更进一步：用你提供的代码仓库对模型做风格微调Pro / Team 更适合「日常使用 ZZW Code 加速 H# 开发」的团队。
MAX+ 的「专属 AI 训练」具体指什么？ +
订阅 MAX+ 后，你可以向我们的训练团队提交你的 H# 代码仓库（公开或私有），我们会基于你的代码风格、命名习惯、惯用模式对底层模型做一次轻量微调。微调后的模型在你自己的项目里会输出明显更地道的 H#，包括你团队特有的工具函数调用约定。提交代码前会签双向 NDA，训练数据物理隔离，训练结束后 90 天内销毁。
教育 / 开源项目有折扣吗？ +
有。GitHub 上有公开仓库的开发者可以申请 Pro 版免费 1 年，在校学生 / 教师可以申请 Team 版 50% 折扣。把仓库地址或学校邮箱发到 hi@hsharp.dev 即可。
